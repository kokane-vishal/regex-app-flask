How to Use This Regex Matching Tool

  1. Test String: Enter the text you want to search through in the "Test String" box.
  2. Regex Pattern: Enter your regular expression in the "Regex Pattern" box.
  3. Click "Find Matches" to see all matching substrings.

Quick Regex Guide:

  • \d - Matches any digit (0-9)
  • \w - Matches any word character (a-z, A-Z, 0-9, _)
  • \s - Matches any whitespace character
  • ^ - Matches the start of a line
  • $ - Matches the end of a line
  • * - Matches 0 or more occurrences
  • + - Matches 1 or more occurrences
  • ? - Matches 0 or 1 occurrence
  • [...] - Matches any single character in brackets

Example: \b\w+@\w+\.\w{2,3}\b matches most email addresses.
